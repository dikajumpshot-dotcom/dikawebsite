Nama saya I Putu Gede Agus Suardika, mahasiswa baru Program Studi Sistem Informasi di Universitas Pendidikan Ganesha. Website ini ruang kecil untuk memperkenalkan diri.

Saya tertarik mempelajari bagaimana teknologi sistem informasi bisa membantu menyelesaikan masalah sehari-hari, dan tidak sabar untuk menimba ilmu lebih dalam selama masa kuliah. Di luar kuliah, saya suka hal-hal santai seperti gaming, musik, dan ngopi bareng teman.
